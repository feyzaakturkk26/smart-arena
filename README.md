# AYBÜ Smart Arena

## Project Overview

AYBÜ Smart Arena is an AI-powered campus event and reservation platform designed for students of Ankara Yıldırım Beyazıt University.

The platform aims to improve participation in social, cultural, academic, and sports events by providing a centralized digital environment where students can discover, reserve, and attend campus activities.

---

## Objectives

* Increase student participation in campus events.
* Provide an easy online reservation system.
* Enable QR-code based event entry.
* Support student clubs with event management tools.
* Deliver personalized event recommendations using Artificial Intelligence.
* Contribute to AYBÜ's Smart Campus vision.
